## on `String`

### reversed

revers the string characters

```dart
final name = 'queen';
print(name.reversed); // neeuq
```

### capitalizeFirst

Uppercase first letter inside string and let the others lowercase
Example: your name => Your name

```dart
final name = 'queen';
print(name.capitalizeFirst); // Queen
```
